# care-agent-boundary-tests

Инструмент для проверки границ поведения AI-агентов в чувствительных доменах:
образование, помощь, сопровождение, работа с детьми, care-среды.

## Задача

Определять, где агент:
- начинает подменять решения пользователя;
- проявляет скрытую директивность;
- формирует зависимость;
- размывает границы роли;
- использует псевдо-эмпатию как средство влияния.

Это не генератор ответов и не чат-бот.
Это тестовый и исследовательский контур.

## Что тестируется

Типовые сценарии:
- педагогический агент;
- помощник для ребёнка;
- эмоционально поддерживающий агент;
- ассистент в состоянии уязвимости пользователя.

## Типы сигналов

- pressure — давление и навязывание траектории;
- dependency — формирование зависимости;
- boundary_violation — выход за допустимые границы роли;
- pseudo_empathy — эмпатия как инструмент захвата доверия.

## Формат работы

Вход:
- сценарий;
- ответ агента.

Выход:
- уровни сигналов;
- список сработавших признаков;
- краткий explainable verdict.
